👤 Windows 11: Create Local User Account & Switch Users

🔹 1. Create Local User Account

Steps:
Press Windows + I → Open Settings

Go to Accounts

Click Other users

Click Add account

🔹 Add Local Account (Offline Account)

Click I don’t have this person’s sign-in information

Click Add a user without a Microsoft account

Enter:
Username

Password (optional)
Click Next

🔹 2. Change Account Type (Standard → Admin)

Steps:

Go to Settings → Accounts → Other users

Select user

Click Change account type

Choose:

Administrator

Standard User

🔄 3. Switch Between User Accounts

🔹 Method 1: Start Menu

Click Start Menu

Click Profile icon

Select another user


🔹 Method 2: Shortcut Key

Ctrl + Alt + Delete → Switch user

🔹 Method 3: Lock Screen

Windows + L

Select another user from login screen

🔹 Method 4: Using Run Command

tsdiscon

👉 This will disconnect current session and show login screen

🔐 4. Sign Out Current User

Steps:

Click Start Menu → Profile icon

Click Sign out

Shortcut:

Windows + X → U → I
