🔹 1. Explore File Explorer Features
Steps:
Press Windows + E
Explore sections:
Quick Access
This PC
Network
OneDrive
Features:
Tabs support
Search bar
Ribbon menu

🔹 2. Create and Manage Folders
Steps:
Open File Explorer
Right-click → New → Folder
Enter name
Shortcut:
Ctrl + Shift + N → New Folder

🔹 3. File Attributes (Read-only, Hidden)
Steps:
Right-click file → Properties
Check:
Read-only
Hidden

🔹 4. NTFS Permissions Lab (Read/Write/Modify)
Steps:
Right-click folder → Properties
Go to Security tab
Click Edit
Assign permissions:
Read
Write
Modify

🔹 5. Ownership of Files/Folders
Steps:
Properties → Security → Advanced
Click Change (Owner)
Select user → Apply

🔹 6. Map and Unmap Network Drives
Map Drive:
Open File Explorer
Click This PC → Map network drive
Enter path (\server\folder)
Unmap:
Right-click drive → Disconnect

🔹 7. Create Disk Partitions
Steps:
Press Windows + X → Disk Management
Right-click unallocated space
Click New Simple Volume

🔹 8. Format a Partition (NTFS vs exFAT)
Steps:
Right-click drive → Format
Choose:
NTFS → Windows system
exFAT → Cross-platform

🔹 9. Extend and Shrink Volumes
Steps:
Open Disk Management
Right-click partition
Select:
Extend Volume
Shrink Volume

🔹 10. Use Disk Cleanup Tool
Steps:
Press Windows + S → Search "Disk Cleanup"
Select drive
Delete temporary files

🔹 11. Enable Storage Sense
Steps:
Go to Settings → System → Storage
Enable Storage Sense

🔹 12. File History Configuration
Steps:
Go to Settings → System → Storage → Advanced backup
Enable File History

🔹 13. Compress and Extract Files
Compress:
Right-click → Send to → Compressed folder
Extract:
Right-click → Extract All

🔹 14. Encrypt Files using EFS
Steps:
Right-click file → Properties
Click Advanced
Enable:
Encrypt contents

🔹 15. Recover Deleted Files
Methods:
Recycle Bin → Restore
File History → Restore
Backup
