⚙️ Windows 11: Settings vs Control Panel & Power Options
🔹 1. Open Settings
Steps:
Press Windows + I
Settings window will open

Alternative Methods:
Press Windows + X → Settings
Search Settings in Start Menu

🔹 2. Open Control Panel
Steps:
Press Windows + R
Type control → Press Enter

Alternative Methods:
Search Control Panel in Start Menu
Press Windows + X → Run → type control

🔹 3. Settings vs Control Panel (Overview)
Feature	Settings	Control Panel
Interface	Modern	Classic
Usage	Daily tasks	Advanced settings
Status	Updated	постепенно removed

👉 Use Settings for daily work
👉 Use Control Panel for advanced configuration

🔐 4. Lock, Sign Out, Restart, Shutdown

🔹 Lock Screen

Steps:

Press Windows + L

Alternative:

Press Ctrl + Alt + Delete → Lock

🔹 Sign Out

Steps:

Press Ctrl + Alt + Delete
Click Sign out

Shortcut:

Windows + X → U → I

🔹 Shutdown

Steps:

Click Start Menu → Power icon
Click Shut down

Shortcuts:

Alt + F4 → Shutdown
Windows + X → U → U

🔹 Restart

Steps:

Click Start Menu → Power icon
Click Restart

Shortcuts:

Windows + X → U → R
Alt + F4 → Restart

🔹 Sleep

Steps:

Start Menu → Power → Sleep

Shortcut:

Windows + X → U → S

⚡ 5. Important Shortcut Keys

🖥️ Basic Shortcuts

Windows + I → Open Settings

Windows + R → Run Command
  
Windows + E → File Explorer

Windows + D → Show Desktop

Windows + S → Search

🔄 Navigation & Control

Alt + Tab → Switch apps

Alt + F4 → Close app / Shutdown menu

Ctrl + Shift + Esc → Task Manager

Windows + Tab → Task View

⚡ Power User Shortcuts

Windows + X → Power Menu

Windows + L → Lock Screen

Windows + Shift + S → Screenshot

Windows + Arrow Keys → Snap windows

🔥 Pro Tips

Use Windows + I for quick settings access

Use Windows + X → U → (key) for fast shutdown/restart

Use Windows + L to secure your PC instantly
