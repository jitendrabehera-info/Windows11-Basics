## 🖥️ Manage Desktop Icons

### 🔹 Step 1: Show/Hide Desktop Icons

1. Right-click on Desktop
2. Click **View**
3. Select **Show desktop icons**

```
<img width="1307" height="807" alt="image" src="https://github.com/user-attachments/assets/b3bbc35f-c6c9-477a-b685-3fc8bf3e293c" />

```

---

### 🔹 Step 2: Add System Icons (This PC, Recycle Bin)

1. Press **Windows + I**
2. Go to **Personalization → Themes**
3. Click **Desktop icon settings**
4. Select:

   * This PC
   * Recycle Bin
   * Network

🖼️ Add Image Here:

```
![System Icons](images/system-icons.png)
```

---

### 🔹 Step 3: Arrange Icons

* Right-click Desktop → **View**

  * Auto arrange icons
  * Align to grid

---

### 🔹 Step 4: Change Icon Size

* Right-click Desktop → View → Select:

  * Large icons
  * Medium icons
  * Small icons

## 🎨 7. Change System Theme & Wallpaper

### 🔹 Step 1: Open Personalization

1. Press **Windows + I**
2. Go to **Personalization**

---

### 🔹 Step 2: Change Theme

1. Click **Themes**
2. Choose any theme:

   * Light
   * Dark
   * Custom

🖼️ Add Image:

```
![Themes](images/themes.png)
```

---

### 🔹 Step 3: Change Wallpaper

1. Go to **Background**

2. Select:

   * Picture
   * Solid color
   * Slideshow

3. Click **Browse** → Select image

🖼️ Add Image:

```
![Wallpaper](images/wallpaper.png)
```

---

### 🔹 Step 4: Change Colors

1. Go to **Colors**
2. Choose:

   * Accent color
   * Transparency effects

---

## 🌍 8. Configure Date, Time, and Region

### 🔹 Step 1: Open Settings

1. Press **Windows + I**
2. Go to **Time & Language**

---

### 🔹 Step 2: Set Date & Time

1. Click **Date & Time**
2. Enable:

   * Set time automatically
   * Set time zone automatically

🖼️ Add Image:

```
![Date Time](images/date-time.png)
```

---

### 🔹 Step 3: Change Time Zone Manually

* Turn OFF automatic → Select your timezone

---

### 🔹 Step 4: Change Region

1. Go to **Language & Region**
2. Select:

   * Country/Region
   * Regional format
