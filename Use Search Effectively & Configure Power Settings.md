🔎 1. Use Windows Search Effectively

🔹 Open Windows Search

Methods:

Press Windows + S

Press Windows key and start typing

Click Search icon on taskbar

🔹 Search for Apps, Files, Settings

Steps:

Open 

Type keyword (e.g., Control Panel, Settings)

Click result

🔹 Filter Search Results

After searching, choose:

Apps
Documents
Web
Settings

👉 Helps find results faster

🔹 Use Advanced Search (File Explorer)

Steps:

Press Windows + E

Use search bar (top-right)

Type:
name:report date:today type:pdf

🔹 Run Commands from Search

Type commands like:

cmd
control
notepad

👉 Press Enter to open directly

⚡ Useful Search Shortcuts

Windows + S → Open Search

Windows + E → File Explorer

Windows + R → Run dialog

Alt + Enter → File properties

🔋 2. Configure Power & Sleep Settings

🔹 Step 1: Open Power Settings

Methods:

Press Windows + I → Settings

Go to System → Power & battery

🔹 Step 2: Configure Screen Timeout

Steps:

Click Screen and sleep

Set time for:
Screen off
Sleep mode

🔹 Step 3: Change Power Mode

Steps:

Under Power mode

Select:
Best power efficiency
Balanced
Best performance

🔹 Step 4: Advanced Power Options

Steps:

Press Windows + R

Type:
powercfg.cpl

Modify:
Power plans
Advanced settings
